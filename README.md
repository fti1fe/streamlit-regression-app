# Laptop Price Prediction App

Ive developed this app for COM763 Advanced Machine Learning.
It predicts laptop prices from hardware and product features using supervised regression.

The Dataset Ive used is available on kaggle:
[laptops_cleaned.csv](https://www.kaggle.com/datasets/githubmasterin/laptops-cleaned-dataset)

Models Ive tested for this:
Dummy Regressor
Ridge Regression
Random Forest Regressor
MLP Regressor

I have deployed the app here:
